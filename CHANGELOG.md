# Changelog

This project uses semantic version tags (`vMAJOR.MINOR.PATCH`) for release
artifacts generated by `make release`.

## Unreleased

- Added global TUI search for projects, merge requests and branches.
- Added local multi-repo workspaces with consolidated MR status and copyable
  workspace summaries.
- Added a responsive three-area TUI shell with header, sidebar/main content,
  footer, theme application and optional mouse support.
- Added cross-platform release packaging and an install script.
